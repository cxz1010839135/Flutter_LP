## 1. 引言

### 1.1 编写目的

本说明书用于介绍「领鹏智能机器人上位机软件」（以下简称「本软件」或「领鹏智能」）的功能、运行环境、安装方法及基本操作方法，供最终用户、现场工程师及软件著作权登记审查使用。

### 1.2 软件背景

本软件是 ____________________（待填写，公司名称）面向 LP 系列工业机器人控制器开发的上位机应用程序。软件通过以太网与机器人控制器建立连接，提供连接管理、状态监控、手动操控、可视化编程、程序运行监视、参数配置等功能，适用于工业自动化生产线的调试、编程与日常运维。

## 2. 运行环境

### 2.1 硬件环境

| 项目 | 最低要求 | 推荐配置 |
|------|----------|----------|
| 处理器 | x64 双核 1.5GHz 或 ARM 四核 | Intel i5 / 工控平板 x64 |

- 运行支撑：WebView2 运行时（Windows 11 通常已内置）

## 3. 安装与卸载

### 3.1 Windows 安装

1. 双击 `LPRobot-V1.7.9-x64.msi` 安装包。  
2. 安装向导为简体中文，可选择安装目录（默认：`C:\Program Files\Lingpeng\领鹏智能\`）。  
3. 安装完成后，开始菜单或桌面将出现「领鹏智能」快捷方式，主程序为 **领鹏智能.exe**。  
4. 安装目录包含：`领鹏智能.exe`、`config\`（配置与默认程序）、`dll\visualprogram\`（Blockly 资源）。  
5. 若安装至 Program Files 等只读目录，用户数据（可写配置、工程文件）自动存放于：  
   `%LOCALAPPDATA%\Lingpeng\LPRobot\`  
6. **无需以管理员身份运行**程序。

### 3.2 Android 安装

1. 将 `LPRobot-V1.7.9.apk` 拷贝至设备或通过 ADB 安装。  
2. 若已安装旧版，建议先卸载再安装新版。  
3. 首次运行需授予存储相关权限（用于读写工程与配置文件）。

### 3.3 卸载

- **Windows**：控制面板 → 程序和功能 → 选择「领鹏智能」→ 卸载。用户数据目录 `%LOCALAPPDATA%\Lingpeng\LPRobot\` 需手动删除（若需清除全部数据）。  
- **Android**：系统设置 → 应用管理 → 领鹏智能 → 卸载。

## 4. 软件概述

### 4.1 软件定位

领鹏智能机器人上位机软件是连接、编程、操控、监控领鹏机器人的统一人机界面（HMI），实现「连接 → 主页总览 → 分模块操作」的工作流。

### 4.2 总体架构

```
┌─────────────────────────────────────────────────────────┐
│                    领鹏智能 上位机软件                      │
├──────────┬──────────┬──────────┬──────────┬─────────────┤
│  连接页   │   主页    │  可视化   │  操控页   │   监控页     │
│ Connect  │   Home   │  Blockly │ Control  │  Monitor    │
├──────────┴──────────┴──────────┴──────────┴─────────────┤
│              核心层：RobotState / HttpManager              │
│              遥测轮询：RobotStatePoller (200ms)            │
├─────────────────────────────────────────────────────────┤
│                    HTTP/JSON ↔ 机器人控制器               │
└─────────────────────────────────────────────────────────┘
```

### 4.3 主界面布局（主页）

连接成功后进入主界面，自上而下分为：

1. **顶栏（位姿栏）**：显示当前 TCP 位姿（X/Y/Z/Rx/Ry/Rz）、运行模式、连接状态。  
2. **中部工作区**：左侧为功能导航（编程、操控、监控、工具）；中央为机器人示意图与状态面板；右侧为运行控制侧栏（启动/停止/速度调节等）。  
3. **IO 面板**：16 路数字输入/输出状态指示。  
4. **底栏**：报警信息、状态日志、版本号。

界面主色调为领鹏品牌橙（#FF7E1A），背景浅灰（#E5E6EA），实时数值为绿色（#00AF29）。

### 4.4 启动流程

1. 启动程序，自动初始化安装目录与用户数据目录。  
2. 若路径初始化失败，显示「应用初始化失败」及错误详情。  
3. 进入连接页，自动加载上次保存的控制器 IP。  
4. 用户输入 IP 并连接，成功后跳转主界面并开始遥测轮询。

## 5. 功能模块说明

### 5.1 连接模块（ConnectPage）

**功能描述：** 建立与机器人控制器的网络连接，验证控制器版本，同步主程序文件。

**主要功能：**

- IP 地址输入与记忆（保存至 `config/app_settings.json`）  
- 一键连接，显示连接进度与结果  
- 连接成功后自动下载/同步 `config/server/main.xml` 与 `main.rp4`  
- 连接客户端标识：`LPZN V1.7.9`  
- Android 支持硬件键盘输入 IP（适配工控平板/模拟器）

**连接成功条件：** 控制器返回 `result == 1`，且 `data.version` 以 `LP` 开头。

### 5.2 主页模块（MainHomePage）

**功能描述：** 连接后的总览与导航中枢。

**主要功能：**

- 实时显示机器人位姿、运行状态、速度百分比  
- 16 路 IO 输入/输出状态实时刷新  
- 报警信息显示与历史日志  
- 运行控制：启动、停止、暂停、复位等（通过右侧运行侧栏）  
- 速度调节：环形速度控件 + 对话框设置速度百分比  
- 功能导航入口：编程（Blockly）、操控、监控、工具（参数配置）  
- 断开连接并返回连接页  

**数据刷新：** 后台 `RobotStatePoller` 每 200ms 轮询控制器状态。

### 5.3 可视化编程模块（BlocklyDemoPage）

**功能描述：** 基于 Google Blockly 的图形化机器人程序编辑器。

**主要功能：**

- 通过内嵌 WebView 加载本地 Blockly 编辑器（`dll/visualprogram/`）  
- 本地 HTTP 服务（Shelf）提供静态资源，避免跨域问题  
- Dart ↔ JavaScript 桥接（`flutter_bound.js`），实现：  
  - 加载/保存 XML 工程  
  - 保存至控制器程序目录（`config/server/`）  
  - 保存至用户工程（`files/projects/{name}/`）  
  - 保存函数库（`files/funlib/`）  
  - 从 XML 库（`files/xml/`）选择已有工程  
- 退出编程时可上传程序至控制器在线运行  

**适用场景：** 非专业用户通过拖拽积木块编写机器人逻辑，无需手写代码。

### 5.4 操控模块（ControlPage）

**功能描述：** 机器人手动点动、轨迹移动与 IO 操作。

**主要功能：**

- **轴选择导航**：笛卡尔坐标（X/Y/Z/Rx/Ry/Rz）、关节轴、扩展轴  
- **点动面板**：按住方向键/按钮进行连续或步进点动  
- **门型移动 / 直线移动**：指定目标位置或相对位移  
- **IO 模块**：选择 IO 模块并读写输出  
- **点库**：打开点库页面，管理、示教、调用运动点位  
- **界面清零**：坐标系清零与轴配置（ClrZeroPage）  
- **速度条**：橙色速度调节条，与主页速度联动  

布局对齐 Android ControlActivity：左轴选 / 中功能区 / 右模式 / 底 IO。

### 5.5 监控模块（MonitorPage）

**功能描述：** 监视机器人主程序运行状态与 PLC 寄存器。

**主要功能：**

- 加载并显示 `main.rp4` 主程序文本  
- 运行行高亮：当前执行行自动滚动至可视区域  
- 程序启停控制  
- D9000 等特殊寄存器状态显示  
- PLC 监视列表（可配置监视项，持久化至 `files/monitor_plc_watch.json`）  
- 寄存器侧栏：查看与调试特殊寄存器  

### 5.6 工具 / 参数配置模块（ConfigFilePage）

**功能描述：** 机器人配置文件管理与驱动参数编辑。

**主要功能：**

- 配置文件列表浏览  
- 配置文件读写与编解码  
- 驱动参数（DPS）查看与修改  
- 与控制器同步配置  

### 5.7 点库模块（PointLibraryPage）

**功能描述：** 管理机器人运动目标点位。

**主要功能：**

- 点位列表表格展示（名称、坐标、备注等）  
- 添加、编辑、删除点位  
- 从当前位姿示教新点  
- 在操控页调用点位执行运动  

### 5.8 驱动调试模块（DriverPage）

**功能描述：** 驱动器级调试与波形监视（技术人员模式）。

**主要功能：**

- 驱动参数面板  
- 地址调试  
- 波形显示  
- 技术模式门禁（防止误操作）  

### 5.9 文件管理模块（FilesPage）

**功能描述：** 机器人端文件的上传、下载与备份。

**主要功能：**

- 文件列表浏览  
- 上传/下载程序与配置  
- 备份与恢复

## 6. 操作指南

### 6.1 首次连接机器人

1. 确认上位机网卡 IP 与控制器在同一网段（如 192.168.11.x）。  
2. 启动「领鹏智能」。  
3. 在连接页输入控制器 IP（默认 192.168.11.11）。  
4. 点击「连接」按钮，等待提示连接成功。  
5. 自动进入主界面，观察顶栏位姿是否刷新。

### 6.2 编辑并下发程序

1. 在主界面点击「编程」进入 Blockly 编辑器。  
2. 拖拽积木块组合逻辑，或从 XML 库打开已有工程。  
3. 点击保存，选择保存至「控制器程序」或「用户工程」。  
4. 退出编程界面，确认上传至控制器。  
5. 返回主页，点击「启动」运行程序。  
6. 进入「监控」页查看运行行与执行进度。

### 6.3 手动点动机器人

1. 确保机器人处于手动/示教模式且安全条件满足。  
2. 主界面进入「操控」。  
3. 左侧选择要运动的轴（如 X 轴）。  
4. 在中央面板按住「+」或「−」进行点动，或使用步进距离。  
5. 通过底栏 IO 面板确认相关气缸/夹爪信号。  
6. 调节右侧速度条控制点动速度。

### 6.4 调节运行速度

1. 在主界面右侧运行侧栏找到速度环控件。  
2. 点击打开速度调节对话框。  
3. 拖动滑块或输入百分比（0～100%）。  
4. 确认后速度下发至控制器。

### 6.5 断开连接

1. 在主界面选择「断开」或返回连接页。  
2. 软件停止遥测轮询并清除连接状态。  
3. 可重新输入 IP 连接其他控制器。

```
{安装根目录}/

```
%LOCALAPPDATA%\Lingpeng\LPRobot\

    ├── xml/                  # Blockly XML 工程库
    ├── projects/             # 用户工程 {name}/{name}.xml

    ├── downloads/            # 下载目录

| 文件 | 用途 |
|------|------|

## 8. 网络通信说明

### 8.1 通信方式

- 协议：HTTP/1.1  
- 数据格式：JSON  
- 基址：`http://{控制器IP}`  

### 8.2 连接请求示例

```json
POST http://192.168.11.11/
Content-Type: application/json

{"command":"connect","data":"LPZN V1.7.9"}
```

### 8.3 主要 API 类别

| 类别 | 示例 command | 说明 |
|------|--------------|------|
| 连接 | connect | 建立会话 |
| 状态 | robotGetCurState | 轮询位姿、IO、报警 |
| 运行 | start / stop / pause | 程序运行控制 |
| 速度 | setSpeedPercent | 设置速度百分比 |
| 点动 | jog / move | 手动运动 |
| 文件 | upload / download | 程序与配置传输 |

具体命令集与 Android 版 HttpManager 保持一致，由 `lib/network/http_manager.dart` 实现。

## 9. 常见问题

### 9.1 连接失败

- 检查网线/Wi-Fi 与 IP 网段  
- ping 控制器 IP 是否通  
- 确认控制器已上电且 HTTP 服务正常  
- 检查防火墙是否拦截  

### 9.2 应用初始化失败（白屏）

- Windows：确认 exe 同级存在 `config/` 与 `dll/visualprogram/`  
- Android：检查存储权限，或清除应用数据后重试  

### 9.3 Blockly 无法加载

- 确认 `dll/visualprogram/` 目录完整  
- Windows 需安装 WebView2 运行时  
- 查看状态日志中的 HTTP 服务启动信息  

### 9.4 安装后找不到程序

- Windows 默认：`C:\Program Files\Lingpeng\领鹏智能\领鹏智能.exe`  
- 开始菜单搜索「领鹏智能」  

### 9.5 位姿不刷新

- 确认已成功连接  
- 检查控制器是否处于可查询状态  
- 重启软件重新连接
