# 软件著作权登记材料

本目录包含「领鹏智能机器人上位机软件 V1.7.9」著作权登记所需文档，可直接打印或转 PDF 后提交中国版权保护中心（CPCC）。

## 文件清单

| 文件 | 用途 |
|------|------|
| [登记信息表.md](./登记信息表.md) / **登记信息表.docx** | 申请表填写参考（著作权人、开发日期等已留空待填） |
| [软件说明书.md](./软件说明书.md) / **软件说明书.docx** | 用户手册 / 设计说明书（建议 ≥10 页，可插入界面截图） |
| [源程序-前30页.txt](./源程序-前30页.txt) / **源程序-前30页.docx** | 鉴别材料：源程序前 30 页（每页 50 行） |
| [源程序-后30页.txt](./源程序-后30页.txt) / **源程序-后30页.docx** | 鉴别材料：源程序后 30 页（每页 50 行） |

## 生成 Word 文档

在工程根目录执行：

```powershell
python tool/generate_copyright_word.py
```

将生成 `docs/copyright/` 下 4 个 `.docx` 文件，含页眉「领鹏智能机器人上位机软件 V1.7.9」与页码。著作权人、公司名称、开发完成日期等字段已标注 **（待填写）**。

## 重新生成源程序

版本号或源码变更后，在工程根目录执行：

```powershell
dart run tool/generate_copyright_source.dart
```

## 提交前检查

1. 在 Word 中搜索「待填写」，补全：著作权人、公司名称、开发完成日期、联系人等。
2. 在 `软件说明书.docx` 中插入连接页、主页、Blockly、操控、监控等界面截图。
3. 导出为 PDF 后上传 CPCC；源程序 docx 确认每页 ≥50 行（等宽字体）。
4. 登录 https://register.ccopyright.com.cn 在线填表并上传 PDF。
