## 三、功能与技术特点（摘要，可直接粘贴到申请表）

**主要功能：**

本软件为领鹏系列工业机器人提供上位机人机交互与编程控制。用户通过以太网连接机器人控制器后，可完成：设备连接与状态监测、实时位姿与 IO 显示、机器人点动与轨迹操控、Blockly 可视化程序编辑与下发、程序运行监控、参数配置与点库管理、配置文件备份等功能。支持 Windows 桌面与 Android 横屏两种部署形态，适用于工业现场调试与生产运维。

**技术特点：**

1. 采用 Flutter 跨平台框架，业务逻辑以纯 Dart 实现，Windows/Android 共用同一套代码基线。
2. 基于 HTTP/JSON 与机器人控制器通信，内网明文协议，连接标识为 `LPZN V{version}`。
3. 集成 Google Blockly 可视化编程，通过本地 HTTP 服务与 WebView 桥接，支持工程保存至 `config/server`、`files/projects`、`files/funlib`。
4. 200ms 周期轮询位姿、IO、报警等遥测数据，主界面实时刷新。
5. Windows 发布为 MSI 安装包（领鹏智能.exe），安装至 Program Files 时用户数据自动迁移至 `%LOCALAPPDATA%\Lingpeng\LPRobot\`，无需管理员权限。

## 七、登记流程简述

1. 登录 [中国版权保护中心](https://register.ccopyright.com.cn) 注册账号。
2. 选择「计算机软件著作权登记」→ 填写申请表（参照上文）。
3. 上传 PDF：软件说明书、源程序前 30 页 + 后 30 页（合并或分文件，按系统要求）。
4. 邮寄或在线提交鉴别材料（以当前 CPCC 政策为准）。
5. 缴费（官费约 300 元/件，加急另计）。
6. 审查周期约 30～60 个工作日（普通），通过后领取登记证书。
