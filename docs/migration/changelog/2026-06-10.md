# 2026-06-10 变更详情（1.6.5）

## 背景

Android 版进入 Blockly 报错或卡在 85%/红屏；Windows 双击 `打包Windows安装包.bat` 出现乱码命令。本版修复移动端 Blockly 加载链路与 Windows MSI 打包脚本。

---

## Android Blockly

| 问题 | 处理 |
|------|------|
| 未找到 `dll/visualprogram` | APK 内嵌 zip，首次打开解压到 `installRoot/dll/visualprogram/` |
| `ERR_CLEARTEXT_NOT_PERMITTED` | `AndroidManifest` + `network_security_config.xml` 允许明文 HTTP |
| WebView 缩左上角、红底花屏 | `displayWithHybridComposition: false` + `SizedBox.expand` + 加载后 `Blockly.svgResize` |

| 文件 | 说明 |
|------|------|
| `lib/blockly/lp_blockly_asset_bootstrap.dart` | 解压与进度回调 |
| `lib/blockly/lp_blockly_webview_visibility.dart` | Android WebView 创建与布局 |
| `tool/sync_blockly_assets.dart` | 工程内 zip 打包 |
| `android/app/build.gradle.kts` | `preBuild` 调用 sync |
| `pubspec.yaml` | `archive` 依赖 + `assets/blockly/visualprogram.zip` |

---

## Windows 打包

| 问题 | 处理 |
|------|------|
| `$ProjectRoot` 为空 | `Resolve-ProjectRoot` + `LPROBOT_PROJECT_ROOT` 环境变量 |
| ps1 注释与代码粘连 | 重写 `windows.ps1`，UTF-8 BOM + CRLF |
| bat 乱码执行 `'燂紝...if'` | `打包Windows安装包.bat` 纯 ASCII，简化退出逻辑 |
| `flutter pub get` 失败 | 默认国内镜像；`Invoke-ExternalCommand` 忽略 stderr 误报 |
| 缺少 zip 资源 | Windows 构建前同样执行 `sync_blockly_assets.dart` |

---
