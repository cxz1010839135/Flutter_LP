## 保存性能（驱控参数 / 配置文件）

| 优化 | 说明 |
|------|------|
| `ConfigFileUploadMode.singleFile` | 单文件保存：**不先删目录**（对齐 Android `postFile`） |
| 临时目录写盘 | 小文件写入 `Directory.systemTemp`，避免走 Backup 深路径 |
| 保存后不重载 | 驱控参数保存成功不再 `getFile` 整文件刷新 |
| 跳过 chmod | `driverparams.dps` 保存后不再额外 `robotChmod` |

向导内其它 `.pa` 保存同样使用 `singleFile` 模式。
