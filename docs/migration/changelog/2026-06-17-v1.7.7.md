# 2026-06-17 变更详情（1.7.7）

## 背景

在 1.7.5 操控页 UI 基础上，继续对齐 Android 主页/底栏/点库体验；统一桌面端字体；解决 Windows 窗口拖动缩放时的留白、裁切与溢出问题。

---

## 一、窗口与布局缩放

### 问题

固定 1280×720 全局视口在窗口比例变化时，难以同时满足「铺满窗口」「等比缩放」「四边不裁切」。

### 方案（最终）

- **移除** `MaterialApp.builder` 中的全局 `LpUniformAppViewport` 固定画布
- 各页面沿用 **Flex / Expanded / LayoutBuilder** 自适应真实窗口尺寸
- 拖动改变宽或高时，内容随窗口伸缩铺满，无四周大块留白，也无顶栏/底栏裁切

### 主页侧栏

- **左侧导航**：四键 `Expanded` 纵向均分；贴图 `BoxFit.fill` 铺满各自格子
- **右侧运行栏**：启动/停止/速度/复位四格 `Expanded` 等分，面板占满列宽

### Windows 原生

- 取消 `WM_SIZING` 16:9 窗口比例锁定（用户可自由拉伸）
- 保留 `WM_GETMINMAXINFO` 最小窗口 960×540

### 保留组件

- `LpUniformAppViewport` / `LpScaledWorkspace` 仍可作为局部视口工具，默认不再包裹全应用

---

## 二、字体

- 内置 **Roboto**（英文/数字）+ **思源黑体 SC**（中文）
- 资源目录 `assets/fonts/`；主题经 `lp_app_fonts.dart` 统一注入
- 移除点库表等处硬编码 `Consolas` / `monospace`

---

## 三、主页 UI

- 左侧四键改用 `config/imgs` 素材：`control_*` / `program_*` / `monitor_*` / `tool_*`
- 同步至 `assets/home/nav/`；`HomeNavButton` 优先读外置 `config/imgs`
- 底栏中间区 **11:1**（机型图 : IO+状态）；IO 左贴、状态右贴，中间约 28px 间距
- 中央机型图按 `robotType` 加载，标题与运行百分比随布局缩放

---

## 四、连接页

- Logo 约 **96×380** 显示区域
- 版本号 **17px**、字重 w600、灰色，与 Logo 间距微调

---

## 五、操控页与点库

- **四轴机型**：关节点动滚轮仅显示 4 轴并铺满可用高度
- **点库表**：四轴时轴列均分铺满；表头与数据列宽对齐；字号 15/14，提升可读性
- 操控页去掉内层 `LpScaledWorkspace`，避免与（已移除的）全局视口双重缩放

---

## 六、构建与安装包

- 根目录 **`nuget.config`**；`LPRobot.Installer.wixproj` 设置 `<NuGetAudit>false</NuGetAudit>`，消除 WiX 构建 NU1900 审计警告
