# 2026-06-17 变更详情（1.7.5）

## 背景

对齐 Android 操控页 UI：点动/门型/直线/IO 面板布局与贴图；统一应用图标为 `ic_launcher`；操控页按设计稿宽度等比缩放。

---

## 一、操控页 UI

### 点动面板（X/Y/Z/关节）

- 中间白框提取为 `ControlFunctionFrame`，三行等分布局
- 速度行：± 按钮 + 橙色滑条 + 速度百分比
- 模式四格：Flutter 自绘（去 PNG 叠层）；高度为行高 2/3，垂直居中
- 关节模式：左侧轴号滚轮在 XYZ 与白框间空隙居中

### 门型 / 直线面板

- 共用 `ControlFunctionFrame` 白框
- 目标点 / 避障高度：标签贴紧、成组居中，输入框对齐，宽度约 66.7%
- 避障高度数字居中，字号随框高动态缩放
- 速度滑条全宽；确定按钮相对白框居中

### IO 面板

- IO 格改用 Android 贴图 `io_g_*` / `io_o_*`（亮/灭）
- IN/OUT 标签贴图 `io_input` / `io_output`
- 放大格子尺寸，缩小行间距，上下留白

### 整页缩放

- `LpScaledWorkspace`：按 1280×720 设计稿 `fitWidth` 等比缩放，最大化时左右贴边

---

## 二、应用图标

- 打包图标统一为 `config/imgs/ic_launcher.png`（`flutter_launcher_icons` 生成 Android/iOS/Windows/macOS）
- 内置资源 `assets/branding/ic_launcher.png`；连接页优先加载
- **主页顶栏**仍使用 `home_top_logo.png`（横条 Logo，不变）

---

## 三、主页与其它 UI

- 左侧导航卡片贴图、底栏 IO 横向布局、启动/停止气泡样式等延续 1.7.4 优化
- 右侧运行侧栏、状态面板布局调整
